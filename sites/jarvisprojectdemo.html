What is the Jarvis Project ?

The jarvis project will be born from the basic idea , of a truth web assistant so friendly as Jarvis (from Iron Man),You don't understand what i'm saying ? Let's make it easy to understand, We want to make a online SIRI that will help you with everyday tasks.

  • Schedule events on a calendar
  • User that have Javascript knowledge will be able to create custom commands
  • A module for send html mails with YOUR email account (Outlook-hotmail-Gmail-zoho <--smtp's) and of course creating it with Voice dictation !
