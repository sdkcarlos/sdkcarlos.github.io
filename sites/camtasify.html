Why a plugin with only snapshots?

If a plugin made with javascript , decides to give support to graphic edition (gamma,gray scale effect) that means the video tag will be rendered on a canvas every x milliseconds. That repercutes in the performance (in a big way !).
You can use camtasify to retrieve a image an then use another plugin of edition or create something to edit the photo by yourself. 

<!-- To add responsive efect, add 100% to both dimensions.
Camtasify is responsive to the parent container
<div style="width:100%;height:100%;" > -->
<div id="container" style="width:400px;height:400px;" >
<script>
$("#container").camtasify({
    onUnsupported:function(){
        console.log("You need to find a solution with flash :(");
    },
    onSuccess:function(camtasifyInstance){
        console.log(camtasifyInstance);
        // Read the docs for more information
        // with the instance you can pause play
        // camtasifyInstance.takeSnapshot(function(DataURL){
        // console.log(DataURL);
        // });
    },
    onError: function(err){
        console.log(err);

        switch(err.name){
            case 'PermissionDeniedError':
                alert("Permission to use a media device was denied by the user or the system.");
            break;
            case 'NotFoundError':
                alert('No media tracks of the type specified were found that satisfy the constraints specified.');
            break;
            case 'InexistentDOMElement':
                alert("Camtasify neeeds an existent div ! None given.");
            break;
            default:
                alert("An unknown error just happened ! :(");
            break;
        }
    }
});
</script>
