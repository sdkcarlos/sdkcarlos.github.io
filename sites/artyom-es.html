Conoce que puede y no puede hacer Artyom

Desafortunadamente (al menos hasta ahora), SpeechRecognition y SpeechSynthesis esta disponible solo para navegadores basados en chromium (webKit ) como Google Chrome o Safari Apple, de todas maneras puedes incluir la libreria de Artyom y dejar a los usuarios que no usan estos exploradores sin ser afectados. incluyendo las herramientas para saber cuando es el caso.

Artyom proporciona soporte completo para 3 lenguajes, otros lenguajes pueden usar Voice recognition sin sintetizador de voz :
English (lang: en-GB,en) Deutsch (lang: de-DE,de) Español (lang: es-ES,es-CO,es)

Pero ... como podemos lograr algo como eso con JAVASCRIPT? Mira el ejemplo : 


// Inicializamos artyom y proporcionamos la configuración básica , el lenguaje
// la propiedad "continuous" le asignamos un false con el fin de que se detenga cuando
// reconozca un solo comando.
// Activamos el debug para ver en la consola que pasa con artyom detras de escena
// (ver que esta entendiendo y cuando son llamados algunos eventos etc)


artyom.initialize({
    lang:"es-CO",
    continuous:false,
    listen:true,//De otra manera solo guardaremos la configuración pero no iniciara el reconocimiento.
    debug:true
});


/**
 * Y en el archivo de comandos (artyomCommands.js) se guardaron los siguientes comandos
 */

var artyomCommands = [
        {
            //indexes se compone de un array que contendrá todas las opciones y maneras en la que el usuario llamará este comando.
            indexes: ['hola','que tal','hola como estas','como esta','Cómo estás'],
            //action es una funcion que será llamada cuando Artyom identifique este comando
            action : function(i){
                var forHowareyou = [
                    'Muy bien y no gracias a ti',
                    'No es de tu incumbencia'
                ];
                
                var forHello = [
                    "Holita",
                    "Hola que hace",
                    'Que honda, todo bien?'
                ];
                
                // var i = contiene el indice en el array de la opcion que fue identificada.
                switch(i){
                    case 0:
                    case 1:
                        var frase = forHello[Math.floor(Math.random() * forHello.length)];
                        artyom.say(frase);
                    break;
                    case 2:
                    case 3:
                    case 4:
                        var frase = forHowareyou[Math.floor(Math.random() * forHowareyou.length)];
                        artyom.say(frase);
                    break;
                }
            }
        }
];

Awesome ! How can i achieve that

For this kind of commands there are 2 simple rules you need to follow.

  • Save the command with a property : smart:true //Artyom needs to know when have to think or not
  • In the indexes of the command, put a * in the string , in order to find out what's the wildcard.
    For example :
    indexes:["Activate * please","Deactivate * please"] 
//Command
{
    indexes:["Activate * please","Deactivate * please"],
    smart:true,
    action:function(i,wildcard,sentence){
        switch(i){
            case 0:
                alert("Activating " + wildcard);
            break;
            case 1:
                alert("Deactivating " + wildcard);
            break;
        }
    }
}

But what if I have a command that is only available in a specific view

Don't worry , you can add commands where you want in the moment that you want!For this feature use artyCommands.extend(); Try with this : "wake me up". 

    
    //This commands will be only available in the view where you did this
    //that means , these commands are not saved in artyomCommands.js file
    
    artyCommands.extend([
        {
            indexes:["another useful command","and another"],
            action:function(i){
                alert("this function were called");
            }
        },
        {
            indexes:["paint this room with a * color"],
            smart:true,
            action:function(i,wildcard,sentence){
                alert("The room will be painted in " + wildcard);
            }
        },
        {
            indexes:["wake me up please"],
            action:function(i){
                alert("Well ... I don't know how to do that, create something for that");
            }
        }
    ]);

Lazy Developers Zone !

Para un asistente automático (siri,cortana, google now mode) inicialice artyom así :
    artyom.initialize({
        lang:"es-CO", // Español
        listen:true, // Comience a escuchar desde ahora
        continuous:true, // No deje de reconocer a menos que se le indique lo contrario
        debug:true // Ver en la consola lo que Artyom entiende y otros mensajes
    });

    o en otro idioma

    artyom.initialize({
        lang:"de-DE", // Artyom in Deutsch
        listen:true, 
        continuous:true,  
        debug:true 
    });
Para crear un boton que active artyom y ejecute un solo comando, use de la siguiente manera : 
    function callThisActionWhenUserClicksButtons(){
        artyom.initialize({
            lang:"es-CO",
            listen:true, // Escuche desde ahora
            continuous:false, // No se ejecute en modo continuo
            debug:true//Ver en la consola etc ...
        });
    }
