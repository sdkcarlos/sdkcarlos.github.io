How to install

Include the plugin on your project

cordova plugin add https://github.com/ourcodeworld/cordova-ourcodeworld-appinstalled.git

Remember you need an active android platform, this plugin only is available on android. Then in your document a global variable will be accesible with OurCodeWorldappinstalled

What offers

  • Know if an app is installed on the device.
  • Open dinamically an app if exists.

check
Verify if an app is installed on the device

Use check method to verify dinamically if an app is installed on the device via its package name (com.mycompany.appname).

// Pretty simple :)
OurCodeWorldappinstalled.check("com.ourcodeworld.ourcodeeditorfree",{
    success: function(app){
        console.info("The app is installed");
        // If you want, you can open it directly without use the open method :
        app.open();
    },
    fail: function(){
        console.info("The app is NOT installed");
    },
    error: function(){
        console.info("An error ocurred while trying to check if the app exists.");
    }
});

We need to execute this code in (or after) the deviceready event.

open
Open an installed application on the device

Open a preinstalled application on the device via its package name.

// Pretty simple :)
OurCodeWorldappinstalled.open("com.ourcodeworld.ourcodeeditorfree",{
    success: function(){
        console.info("The app has been started");
    },
    error: function(){
        console.info("The app cannot be started");
    }
});

We need to execute this code in (or after) the deviceready event.
