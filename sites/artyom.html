Before continue
Know what artyom can or cannot do.

Unfortunately, SpeechRecognition and SpeechSynthesis is only available for Web Browsers based in chromium (webKit) like Google Chrome or Safari Apple, anyway you can include artyom files and leave the users that doesn't use one of them unnafected including the tools to know when is available or not.

Let's get started

For the first demo of artyom, you can say hallo or ask how are you.
So ... How can we do something like the demo with JAVASCRIPT? Well then !


// We prepare artyom and give the basic config ,the language, set the continuos to false in order
// to stop when a command is executed and set debug to true (with this you can see what
// artyom understand when you talk and other useful information)


artyom.initialize({
    lang:"en-GB",
    continuous:false,
    debug:true,
    speed:0.9
});


/**
 * And in the commands file the structure is the following
 */

var artyomCommands = [
    {
        //indexes is an array in which you can enter all the ways the user can say to trigger it
        indexes: ['hello how are','how are you','hello'],
        //what should do artyom when the user calls this command
        action : function(i){
            var forHowareyou = [
                'Thankfully alive and still somewhat young and healthy, in this economy what more can I ask for?',
                'Cool as a cucumber',
                'I am doing so fabulous today! I can hardly control myself from dancing.',
                'From what I hear, I am very good.',
                "I can't complain... I've tried, but no one listens.",
                "As long as I can keep the kitten I found today, I'll be fine!"
            ];

            var forHello = [
                "Ring a ding ding, you're talking to the king.",
                "Hi, is John there?",
                'How goes it?'
            ];

            // var i = the index in the array of the given options.
            if((i == 0) || (i == 1)){
                var frase = forHowareyou[Math.floor(Math.random() * forHowareyou.length)];
                artyom.say(frase);
            }else if(i == 2){
                var frase = forHello[Math.floor(Math.random() * forHello.length)];
                artyom.say(frase);
            }
        }
    }
];
